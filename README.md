Project Overview — Bookify

Bookify — turistik evlər və otellər üçün elan saytıdır.
Layihənin məqsədi istifadəçilərə rahat və təhlükəsiz şəkildə səyahət məkanlarını kəşf etmək və təcrübələrini paylaşmaq imkanı yaratmaqdır.
Platforma həm istifadəçilər, həm də ev/otel sahibləri üçün nəzərdə tutulub və hər iki tərəfin ehtiyaclarını qarşılayan sistem təqdim edir.

Bookify is a listing platform for tourist homes and hotels.
The goal of the project is to provide users with a convenient and secure way to discover travel destinations and share their experiences.
The platform is designed for both users and property/hotel owners, offering a system that meets the needs of both sides.

* The project is open for further development
